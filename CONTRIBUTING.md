# Contributing

## Getting Started

1. Begin by forking the repo on github.
2. Cloning it into your local machine.
3. Create a new branch.
4. Make your changes.
5. Commit your changes.
6. Push your branch and initiate a pull request!

## Code Style
- Please follow the existing code style
- Use clippy and a rust LSP to make development easier, don't ignore any clippy warnings.

## Reporting Bugs
Open an issue and include:
- Your Operating System and Amnosia version.
- Steps to reproduce your bug
- The behaviour you expected

## Feature Requests
Open an issue, and describe the feature. If it's good enough then your feature will get implemented!
